# 介绍

RoyceUI 是一套基于Vue2.X的不常用、轻量级组件库，由个人维护。

# 特性

  • 质量可靠

每个组件都有充分单元测试，为后续集成提供保障。

  • 体验极致

以迅速响应、动画流畅为目标，在交互体验方面追求极致。

  • 扩展性强

支持按需引入，轻量灵活。扩展性强，可以方便地基于现有组件实现二次开发。
